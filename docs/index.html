Component Categories

Data Display Forms & Input Navigation Layout Feedback POS/Retail Manufacturing HR/Workforce Media Charts

<erx-data-grid>

Full-featured data grid with sorting, filtering, pagination, and selection

Interactive calendar with events, multiple views, and date selection

<erx-cart>

Shopping cart component for POS and e-commerce applications

<erx-chart>

Versatile charting component supporting line, bar, pie, and more

Payment processing interface for multiple payment methods

Printable receipt component with customizable layout

Camera-based barcode and QR code scanner

<erx-signature-pad>

Touch-enabled signature capture component

<erx-rich-text>

WYSIWYG rich text editor

On-screen keyboard for kiosk and touch applications
