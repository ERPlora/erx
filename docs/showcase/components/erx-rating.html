Props

Prop Type Default Description
value number 0 Current rating value
max number 5 Maximum rating value (number of stars)
allowHalf boolean false Allow half-star ratings
readonly boolean false Read-only mode (no interaction)
disabled boolean false Disabled state
size 'sm' | 'md' | 'lg' 'md' Size variant
color 'primary' | 'secondary' | 'warning' | 'success' | 'danger' 'warning' Color theme
showLabel boolean false Show rating count label
label string '' Custom label text
showValue boolean false Show numeric value next to stars
reviewCount number 0 Number of reviews (displayed if showLabel is true)
icon 'star' | 'heart' | 'circle' | 'thumb' 'star' Icon type

Events

Event Description Payload
erxChange Emitted when rating value changes { value: number, maxValue: number }
erxHover Emitted when user hovers over a rating number

Usage Examples

<!-- Basic rating -->
<erx-rating value="4"></erx-rating>

<!-- Half stars -->
<erx-rating value="4.5" allow-half></erx-rating>

<!-- With label and review count -->
<erx-rating
  value="4.2"
  allow-half
  readonly
  show-label
  review-count="256">
</erx-rating>

<!-- Different icon and color -->
<erx-rating
  value="5"
  icon="heart"
  color="danger"
  size="lg">
</erx-rating>

<!-- Listen to changes -->
<erx-rating
  id="my-rating"
  value="0">
</erx-rating>

<script>
  const rating = document.getElementById('my-rating');
  rating.addEventListener('erxChange', (e) => {
    console.log('New rating:', e.detail.value);
  });
</script>
