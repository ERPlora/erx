Props

Prop Type Default Description
slides array [] Array of slide objects
slides-per-view number 1 Visible slides count
space-between number 0 Space between slides (px)
autoplay boolean false Enable autoplay
autoplay-interval number 5000 Autoplay interval (ms)
loop boolean false Enable infinite loop
effect 'slide' | 'fade' 'slide' Transition effect
centered boolean false Center active slide
show-arrows boolean true Show navigation arrows
show-pagination boolean true Show pagination dots
show-thumbnails boolean false Show thumbnail navigation
show-captions boolean false Show slide captions

Events

Event Description
erxSlideChange Emitted when active slide changes
erxSlideClick Emitted when a slide is clicked
