erx-banner

A notification banner component for displaying important messages at the top of a page or section. Supports various types, dismissible actions, and custom icons.

Basic Banners

This is an informational banner with important updates. Your changes have been saved successfully! Your subscription will expire in 3 days. Unable to connect to the server. Please try again later.

Dismissible Banners

Click the X button to dismiss this banner. This warning can be dismissed by the user.

With Actions

A new version is available with exciting features. Your plan is limited. Upgrade for more features. Connection failed. Click retry to try again.

With Custom Icons

New features have been launched! Your account is secure and verified.

Sticky Banner

Sticky banners stay at the top of the viewport when scrolling.

This banner will stick to the top when scrolling (in a real page context).

Props

Prop Type Default Description
type 'info' | 'success' | 'warning' | 'error' 'info' Banner type/color scheme
dismissible boolean false Show close button
icon string auto Custom icon name
action-text string - Text for action button
sticky boolean false Stick to top of viewport

Events

Event Description
erxDismiss Emitted when banner is dismissed
erxAction Emitted when action button is clicked
