erx-state

A component for displaying empty states, error states, and placeholder content. Provides visual feedback when there's no data, an error occurred, or content is loading.

Props

Prop Type Default Description
type string 'empty' State type (empty, error, no-data, not-found, offline, access-denied, maintenance)
title string - State title text
description string - State description text
icon string auto Custom icon name
illustration string - Illustration name instead of icon
action-text string - Primary action button text
secondary-action-text string - Secondary action button text
compact boolean false Use compact layout

Events

Event Description
erxAction Emitted when primary action is clicked
erxSecondaryAction Emitted when secondary action is clicked
