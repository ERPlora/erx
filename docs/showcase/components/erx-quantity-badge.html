Props

Prop Type Default Description
value number 0 Quantity value to display
max number 99 Maximum value before showing "+"
size 'sm' | 'md' | 'lg' 'md' Badge size variant
color 'primary' | 'secondary' | 'success' | 'warning' | 'danger' 'primary' Badge color theme
position 'top-right' | 'top-left' | 'bottom-right' | 'bottom-left' 'top-right' Badge position on parent

Usage Example

<div style="position: relative;">
  <button>Cart</button>
  <erx-quantity-badge
    value="12"
    color="danger"
    size="md"
    position="top-right">
  </erx-quantity-badge>
</div>
