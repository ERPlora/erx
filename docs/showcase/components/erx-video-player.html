Props

Prop Type Default Description
src string - Video source URL
poster string - Poster image URL
title string - Video title
autoplay boolean false Auto-play on load
muted boolean false Start muted
loop boolean false Loop video
show-playback-speed boolean false Show speed selector
show-quality-selector boolean false Show quality options
show-pip boolean false Show PiP button
show-chapters boolean false Show chapters markers
compact boolean false Compact controls
minimal boolean false Minimal controls
color string primary Accent color

Events

Event Description
erxPlay Emitted when playback starts
erxPause Emitted when playback pauses
erxEnded Emitted when video ends
erxTimeUpdate Emitted on time updates
erxFullscreenChange Emitted on fullscreen toggle
erxQualityChange Emitted when quality changes
