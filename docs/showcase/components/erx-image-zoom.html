erx-image-zoom

An image zoom viewer component for displaying images with pan and zoom capabilities. Perfect for product images, maps, and detailed views.

Hover Zoom

Hover over the image to see a magnified view.

Click to Zoom

Click on the image to toggle zoom mode.

Magnifier Lens

A circular magnifier follows your cursor.

Side Panel Zoom

Zoomed view appears in a side panel.

Scroll Zoom

Use mouse wheel to zoom in and out, drag to pan.

Different Zoom Levels

2x Zoom
3x Zoom
4x Zoom

Product Image Example

Props

Prop Type Default Description
src string - Image source URL
zoom-src string src High-res image for zoom
mode 'hover' | 'click' | 'lens' | 'side' | 'scroll' 'hover' Zoom interaction mode
zoom-level number 2 Zoom magnification level
lens-size number 100 Lens diameter (lens mode)
side-panel-id string - ID of side panel element
min-zoom number 1 Minimum zoom level (scroll)
max-zoom number 4 Maximum zoom level (scroll)
show-controls boolean false Show zoom controls

Events

Event Description
erxZoomChange Emitted when zoom level changes
erxZoomStart Emitted when zoom starts
erxZoomEnd Emitted when zoom ends
