Props

Prop Type Default Description
employeeId string '' Employee identifier
name string '' Employee full name
title string '' Job title
department string '' Department name
email string '' Email address
phone string '' Phone number
avatar string '' Avatar image URL
variant 'default' | 'compact' | 'inline' 'default' Card layout variant
status 'available' | 'busy' | 'away' | 'offline' 'offline' Online status
showStatus boolean false Show status indicator
showActions boolean false Show quick action buttons
showDetails boolean false Show extended details
clickable boolean false Make card clickable

Events

Event Description
erxClick Emitted when card is clicked
erxEmail Emitted when email action clicked
erxCall Emitted when call action clicked
erxMessage Emitted when message action clicked
erxSchedule Emitted when schedule action clicked
