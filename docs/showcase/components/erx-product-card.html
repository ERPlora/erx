Props

Prop Type Default Description
product ErxProduct required Product data object
size 'sm' | 'md' | 'lg' 'md' Card size variant
orientation 'vertical' | 'horizontal' 'vertical' Card layout orientation
showAddButton boolean true Show add to cart button
showQuantity boolean false Show quantity controls
showStock boolean true Show stock indicator
selected boolean false Mark as selected
disabled boolean false Disable the card
currency string '$' Currency symbol

Events

Event Description

erxQuantityChange Emitted when quantity changes
