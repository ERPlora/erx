Props

Prop Type Default Description
status string 'online' Status value (online, offline, busy, away, pending, success, warning, error, info)
label string - Optional label text
size 'small' | 'medium' | 'large' 'medium' Indicator size
variant 'dot' | 'badge' | 'badge-outlined' 'dot' Display variant
pulse boolean false Show pulse animation
tooltip string - Tooltip text on hover
