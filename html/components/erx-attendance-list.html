erx-attendance-list

Employee attendance tracking list with status indicators, time entries, and filtering capabilities for HR management.

Props

Prop Type Default Description
entries AttendanceEntry[] [] Array of attendance records
view 'daily' | 'weekly' | 'monthly' 'daily' Display view mode
showDateFilter boolean false Show date range filter
showStatusFilter boolean false Show status filter dropdown
showSummary boolean false Show attendance summary statistics
showExport boolean false Show export button
showPrint boolean false Show print button
loading boolean false Show loading state

Events

Event Description
erxEntryClick Emitted when an attendance entry is clicked
erxDateChange Emitted when date filter changes
erxStatusFilter Emitted when status filter changes
erxExport Emitted when export is requested
erxPrint Emitted when print is requested
