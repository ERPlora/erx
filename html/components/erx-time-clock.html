erx-time-clock

Time clock widget for employee clock in/out functionality with real-time display, break tracking, and shift status.

Basic Time Clock

<erx-time-clock employee-id="EMP001" employee-name="John Doe" ></erx-time-clock> <script> const clock = document.querySelector('erx-time-clock'); clock.addEventListener('erxClockIn', (e) => { console.log('Clocked in:', e.detail.timestamp); }); clock.addEventListener('erxClockOut', (e) => { console.log('Clocked out:', e.detail.timestamp); }); </script>

Props

Prop Type Default Description
employeeId string '' Employee identifier
employeeName string '' Employee display name
status 'clocked-out' | 'clocked-in' | 'on-break' 'clocked-out' Current clock status
clockInTime string null Time of clock in (HH:mm format)
showBreaks boolean false Show break start/end buttons
compact boolean false Use compact display mode
requireLocation boolean false Require GPS location for clock in/out
showLocation boolean false Display current location
disabled boolean false Disable all interactions

Events

Event Description
erxClockIn Emitted when employee clocks in
erxClockOut Emitted when employee clocks out
erxBreakStart Emitted when break starts
erxBreakEnd Emitted when break ends
erxLocationError Emitted when location cannot be obtained
